El programa necesita como dato de entrada un arreglo bidimensional de numeros enteros, como proceso principal necesitamos que el programa recorra la 
matriz mediante estructuras principales anidadas, durante este proceso se debe evaluar cada elemento para definir si su valor es par, en caso de que 
lo sea se debe concatenar a una cadena de texto manteniendo el mismo orden de antes, no debemos crear nuevos arreglos, solo debemos procesar la informacion
recibida como parametro, el resultado que se espera es una cadena que muestre todos los valores pares.
